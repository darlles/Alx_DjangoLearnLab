# Social Media API (Django + DRF + Token Auth)

## Overview
A foundational API for a social media application, featuring a custom user model, token-based authentication, profile management, and follow/unfollow mechanics.

## Stack
- Django
- Django REST Framework
- DRF Token Authentication
- SQLite (dev), Pillow for image uploads
